WHAT IT DOES

Everything. Live.

Six panels, one screen, zero configuration. Just run it.

CPU
PROCESSOR
Overall usage, per-core bars, clock frequency, and a 60-second sparkline. Badges flip to WARN or CRIT as load climbs.
MEM
MEMORY
RAM and swap in GB, load average, and sparkline history. Swap is tracked separately so nothing hides.
DSK
DISK
Per-mount usage bars for every drive — C:, D:, /, /home. Total bytes read and written since boot.
NET
NETWORK
Per-interface stats — WiFi, Ethernet, Loopback. Live speeds, IP address, packet counts. Press TAB to switch.
PRC
PROCESS KILLER
Top 20 processes by CPU. Arrow keys to select, K to kill — with a Y/N confirmation so nothing gets killed by accident.
LOG
SYSTEM LOG
Live event log. Auto-warns when CPU or RAM exceeds 90%. Kill events, errors, and startup info — all color-coded.

GETTING STARTED

Install & Run

Two commands on any OS. Or grab the EXE for Windows — no Python needed.

🪟 WINDOWS

# Install dependencies
pip install psutil windows-curses
# Run
python sysmon.py

Or download sysmon.exe — runs on any Windows 10/11 PC with no Python required.

🐧 LINUX / MACOS

# Install dependency
pip install psutil
# Run
python3 sysmon.py

curses is built into Python on Linux and macOS — no extra install needed.

📦 BUILD EXE (Windows)

# Build a standalone executable
pip install pyinstaller
pyinstaller --onefile --console sysmon.py
# Output: dist\sysmon.exe

Share dist\sysmon.exe with anyone. Run it from Command Prompt — not by double-clicking.

REFERENCE

Panel Docs

What each panel shows and how to read it.

CPU

  • Overall CPU % with NOMINAL / WARN / CRIT badge
  • Per-core bars — one row per logical core
  • Current clock frequency in GHz
  • 60-second sparkline history
  • Badge turns amber above 65%, red above 85%

MEMORY

  • RAM used vs total displayed in GB
  • RAM and swap usage bars
  • System load average on Linux/macOS
  • 60-second sparkline history
  • High swap = memory pressure warning

DISK

  • Usage bar for every mounted partition
  • Shows mount point, used %, used GB, total GB
  • Total bytes read and written since boot
  • Color shifts amber at 65%, red at 85%

NETWORK

  • Press TAB to cycle between interfaces
  • Interface name, type (WiFi / ETH / Loop), and IP
  • Live upload and download speed
  • Total bytes sent and received
  • All-interfaces combined summary below divider
  • 60-second download sparkline

PROCESSES

  • Top 20 processes sorted by CPU usage
  • Columns: Name, PID, CPU%, MEM%, Status
  • Use ↑/↓ to highlight a row
  • Press K — popup asks Y to confirm or N to cancel
  • System processes may be protected (Access Denied logged)

SYSTEM LOG

  • Live timestamped event stream
  • Gold = info, amber = warning, red = error
  • Auto-logs CPU or RAM alerts above 90%
  • Logs process kills, errors, and startup info
  • Keeps last 200 log lines in memory
